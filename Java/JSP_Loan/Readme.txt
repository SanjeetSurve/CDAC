Short Project Description:
This project is a Loan Calculation and Management System where users can calculate loan details such as EMI, tax, and discounts for different loan types (i.e., Home Loan & Personal Loan). 
It includes a frontend interface built with JSP, Bootstrap, and CSS, and backend logic in Java. 
Users input loan parameters, and the application calculates the results dynamically, displaying a detailed loan summary.


Commands for running code

Command 1: chmod u+x run
Command 2: ./run
